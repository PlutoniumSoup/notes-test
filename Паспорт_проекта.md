---

## Описание продукта проекта (функциональные возможности)

### 1. Работа с заметками
- Создание и редактирование заметок в окне ввода.
- История версий заметок с возможностью просмотра изменений и отката.
- Автоматическое сопоставление заметки с базой знаний.

### 2. Поиск и сопоставление в базе знаний
- Поиск узлов знаний по ключевым словам и смысловому совпадению.
- Создание новых узлов в графе при уникальности информации.
- Автоматическое связывание новых заметок с существующими узлами.

### 3. Визуализация графа знаний
- Отображение узлов и связей на интерактивном поле.
- Масштабирование и перемещение графа.
- Фильтрация узлов по темам, тегам или важности.
- Подсветка узлов, связанных с текущей заметкой.

### 4. Взаимодействие с узлами графа
- Просмотр подробного описания узла.
- Просмотр всех связей узла с другими элементами.
- Ручное добавление новых связей.
- Аннотации к связям (описание типа взаимосвязи).

### 5. Работа с пробелами знаний
- Отметка узлов как «Здесь есть что изучить» при неполной информации.
- Подсказки для расширения знаний (похожие темы, связанные материалы).
- Список узлов с пробелами для последующего изучения.

### 6. Пользовательский опыт и интерфейс
- Удобный текстовый редактор для ввода заметок.
- Поиск по заметкам и узлам.
- Группировка заметок в коллекции / темы.
- Система тегов и категорий: создание, редактирование, иерархия тегов.
- Фильтрация узлов графа по темам, тегам или важности.

### 7. Дополнительные функции (MVP+ / перспективные)
- Экспорт заметок и графа (PDF, Markdown, JSON).
- Импорт данных из других заметочников.
- Совместная работа: шаринг графа или коллекций с настройкой прав доступа.
- Интеграция с внешними источниками знаний (Wikipedia, научные базы) — перспектива.
- Рекомендательная система («что изучить дальше»).

## Общее описание принципа работы
Пользователь ведёт заметки в окне ввода.  
Система анализирует текст и находит в базе знаний соответствующие узлы, отображая их в виде графа. Пользователь может взаимодействовать с узлами: просматривать связи, открывать подробное описание, добавлять новые связи вручную.  

Если информация в узле неполная, он помечается как «Здесь есть что изучить».  
Рёбра графа содержат описание характера взаимосвязей между узлами знаний.
